# App Overview

## Working Name

Driver Rulebook

The final public app name may change later after App Store / Google Play name checks and trademark review.

## Purpose

Driver Rulebook is a plain-English mobile reference app for truck drivers, shipping workers, receiving workers, warehouse workers, and hazmat-adjacent workers who need quick explanations of confusing trucking safety topics.

The app explains topics such as:

- Hours of Service clocks
- 30-minute breaks
- 14-hour windows
- 60/70-hour recap
- DOT placards
- numbered placards
- bulk vs non-bulk packaging
- GHS pictograms
- SDS sections
- GHS vs DOT differences
- common dock and driver confusion

## Product Positioning

Driver Rulebook is not an ELD, not a legal compliance system, and not official training.

It is a quick reference and guided explanation app.

## Core User Need

Users need fast, clear explanations without reading dense government pages, random forums, outdated PDFs, or relying only on another driver’s app.

## Product Promise

Open the app, search a topic, and understand the issue in plain English.

## Core Principle

Open. Search. Understand.

## Version 1 Scope

Version 1 is local-first and does not require:

- account creation
- sign in
- backend
- user database
- cloud sync

The app should work immediately after download.

## Main Features

- Chapter/page guide
- Hamburger menu with chapters and pages
- Search
- Listen Mode
- Saved topics
- Checklists
- Glossary
- Sources and last reviewed dates
