# Driver Rulebook

Driver Rulebook is the working name for this project. The final App Store / Google Play name may change later.

Driver Rulebook is a local-first mobile reference app for plain-English trucking safety guidance, including Hours of Service explanations, hazmat placard guidance, numbered placards, GHS pictograms, SDS basics, checklists, glossary terms, and common driver/dock situations.

## Core Principle

Open. Search. Understand.

No account required. No sign-in required. No backend required for version 1.

## Planned Platform

Version 1 will be built as a mobile app using React Native with Expo and TypeScript.

React + Vite may be used later only for a marketing/docs website. It is not the main app structure.

## Version 1 Goals

- Local guide content
- Searchable chapters/pages
- Hamburger chapter menu
- Bottom navigation
- Listen Mode using device text-to-speech
- Saved topics
- Dark-first modern UI
- No authentication
- No cloud account
- No backend

## Main Sections

- Quick Start
- HOS / Driver Hours
- Hazmat / Placards
- GHS / SDS
- Common Situations
- Checklists
- Glossary
- Sources / Last Updated
- Settings

## Important Disclaimer

This app is an educational reference only.

It is not:

- legal advice
- official DOT/FMCSA/PHMSA/OSHA training
- an FMCSA-certified ELD
- a replacement for shipping papers
- a replacement for company safety procedures
- a final legal compliance determination tool

Users must confirm requirements with current regulations, shipping papers, SDS documents, carrier procedures, and qualified safety personnel.
